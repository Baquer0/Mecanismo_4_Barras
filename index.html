Introducción

Este proyecto contiene el análisis, simulación y gráficos de un mecanismo de manivela-biela-corredera vertical.

Aquí podrás encontrar el código MATLAB y resultados de análisis realizados.

Código Ejemplo

A continuación un fragmento representativo del código MATLAB para graficar la segunda configuración:

% Parámetros
a = 5; b = 12; theta2 = 305;
% Cálculo punto A
xA = a * cosd(450 - theta2); yA = a * sind(450 - theta2);
% ...
